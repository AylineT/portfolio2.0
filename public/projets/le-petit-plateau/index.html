Le petit plateau

Refont de site web

Octobre 2021

Groupe de 5

Dans le cadre de l'atelier Design Interactif, nous devions proposer une refonte d'un site web existant. Nous avons choisi le site de compagnie de théâtre bordelaise Le petit plateau. Grâce à un audit complet du site, nous avons identifié les élements à modifier, ajouter et supprimer puis nous avons réalisé une maquette. Nous avons développé le site en HTML et SCSS avec Jekyll.


J'ai participé à la création des maquettes sur Figma et au développement pour la version desktop et la version responsive.

Outils utilisés : Figma, Github, Visual Studio Code, Jekyll, Trello
